# 日志规范
使用 Microsoft.Extensions.Logging 库进行日志记录，确保日志记录的可维护性和可扩展。

## 日志级别

| 级别 | 使用场景 |
|------|----------|
| `Debug` | 开发调试信息 |
| `Info` | 正常业务流程 |
| `Warning` | 潜在问题但不影响功能 |
| `Error` | 操作失败需要调查 |
| `Fatal` | 系统级严重错误 |

### 不记录
- 调试用的临时日志
- 敏感信息（密码、Token）
- 大数据量的请求/响应

## 格式

```
[时间戳] [级别] [模块名] 消息内容
```
