# LocalMusicPlayer PRD

## 1. 项目概述

- **项目名称**：LocalMusicPlayer
- **类型**：桌面应用（本地音乐播放器）
- **核心功能**：扫描本地文件夹、管理播放列表、支持多种音频格式的跨平台音乐播放
- **目标用户**：需要播放本地音乐文件的用户

## 2. 技术栈

| 组件 | 技术 |
|------|------|
| UI 框架 | Avalonia 11.3.8 |
| MVVM | ReactiveUI 11.3.8 |
| 音频播放 | LibVLCSharp 3.9.6 |
| 元数据读取 | TagLibSharp 2.3.0 |
| 目标平台 | Windows / macOS / Linux |

## 3. 功能需求

### 3.1 音乐扫描

- 用户选择本地文件夹，递归扫描所有支持的音频文件
- 自动读取文件元数据（标题、艺术家、专辑、时长）
- 支持格式：MP3, FLAC, WAV, AAC, OGG, M4A

### 3.2 播放控制

- 播放 / 暂停 / 停止 / 上一曲 / 下一曲
- 进度条显示和拖拽跳转
- 音量调节（静音 / 增减）
- 双击歌曲立即播放

### 3.3 播放列表

- 显示歌曲列表（标题 / 艺术家 / 时长）
- 当前播放高亮
- 支持键盘上下选择

### 3.4 搜索

- 实时搜索歌曲（按标题、艺术家模糊匹配）
- 列表动态过滤
- 支持高级搜索（按专辑、时长范围、文件格式等）
- 搜索结果高亮显示
- 搜索历史记录

### 3.5 播放队列

- 临时播放队列（不影响当前播放列表）
- 下一首播放（插入到队列顶部）
- 队列管理（拖拽排序、清空、移除）
- 保存队列为播放列表

### 3.6 书签与断点续播

- 自动记录上次播放位置
- 手动添加书签（长音频/有声书支持）
- 书签管理（查看、跳转、删除）
- 跨会话恢复播放进度

## 4. 高级功能

### 4.1 播放模式
- **正常播放**：按顺序播放歌曲
- **随机播放**：随机打乱播放顺序
- **单曲循环**：重复播放当前歌曲
- **列表循环**：循环播放整个播放列表

### 4.2 播放历史与收藏
- 记录最近播放的歌曲历史（最多50首）
- 支持收藏/取消收藏歌曲
- 创建"我的收藏"智能播放列表
- 显示收藏歌曲数量统计

### 4.3 均衡器与音效
- 预设音效模式（流行、摇滚、古典、爵士等）
- 自定义10段均衡器调节
- 低音增强开关
- 环绕声效果选项

### 4.4 歌词显示
- 自动加载LRC格式歌词文件
- 在线歌词搜索与下载
- 歌词同步滚动显示
- 歌词字体大小和颜色自定义
- 无歌词时显示专辑封面

### 4.5 专辑封面管理
- 从音频文件元数据提取专辑封面
- 自动从网络下载缺失的专辑封面
- 手动上传/更换专辑封面
- 专辑视图展示（网格布局）
- 封面缓存管理

### 4.6 多播放列表管理
- 创建/删除/重命名播放列表
- 拖拽歌曲到播放列表
- 播放列表导入/导出（M3U/PLS格式）
- 智能播放列表（基于规则自动生成）
- 播放列表排序（按标题、艺术家、添加时间等）

### 4.7 睡眠定时器
- 设置自动停止播放时间（15/30/45/60分钟或自定义）
- 淡出效果选项
- 倒计时显示
- 定时完成后自动关闭应用选项

### 4.8 快捷键支持
- **空格键**：播放/暂停
- **Ctrl+左/右箭头**：上一曲/下一曲
- **Ctrl+上/下箭头**：音量增加/减少
- **Ctrl+M**：静音切换
- **Ctrl+F**：聚焦搜索框
- **Ctrl+S**：打开设置
- **Delete**：从播放列表移除选中歌曲
- **Ctrl+A**：全选歌曲

### 4.9 通知与系统集成
- 系统托盘图标支持
- 播放状态桌面通知（Windows/macOS/Linux）
- 媒体键支持（播放/暂停/上一曲/下一曲）
- Windows任务栏进度显示
- macOS Dock菜单集成

### 4.10 主题与个性化
- 深色/浅色主题切换
- 自定义主题色
- 界面字体大小调整
- 紧凑/舒适视图模式
- 背景模糊效果选项

### 4.11 文件管理
- 拖放文件/文件夹到窗口添加音乐
- 批量编辑歌曲元数据（标题、艺术家、专辑等）
- 文件格式转换（基础支持）
- 重复歌曲检测与清理
- 文件位置变更自动更新路径

### 4.12 数据统计与可视化
- 音乐库统计（总歌曲数、总时长、艺术家数量等）
- 播放次数统计
- 最常听歌曲/艺术家排行
- 听歌时间分布图表
- 音乐品味分析报告

### 4.13 跨设备同步（未来扩展）
- 播放列表云同步
- 播放进度同步
- 收藏歌曲同步
- 设置备份与恢复

### 4.14 辅助功能
- 屏幕阅读器支持
- 高对比度模式
- 键盘导航优化
- 可访问性标签完善

## 5. 非功能需求

### 5.1 性能要求
- 启动时间 < 3 秒
- 扫描1000首歌曲 < 10 秒
- 播放过程无卡顿、无爆音
- 内存占用 < 200MB（常规使用）
- CPU占用 < 5%（播放时）

### 5.2 兼容性
- 支持拖放文件夹到窗口添加音乐
- 支持Windows 10/11、macOS 10.15+、Ubuntu 20.04+
- 支持Hi-Res音频（24bit/192kHz）
- 支持Unicode文件名和元数据

### 5.3 可靠性
- 异常文件自动跳过，不影响整体扫描
- 播放错误自动尝试下一首
- 配置文件损坏自动恢复默认值
- 定期自动保存播放列表和设置

### 5.4 安全性
- 仅读取音频文件，不修改原始文件（除非用户明确操作）
- 不收集用户隐私数据
- 本地数据存储加密（可选）
- 网络请求仅用于歌词/封面下载，用户可控

## 6. 技术实现细节

### 6.1 数据库设计（可选）
- **轻量级SQLite数据库**存储：
  - 歌曲元数据缓存
  - 播放历史记录
  - 收藏歌曲列表
  - 自定义播放列表
  - 用户设置
  - 歌词缓存
  - 专辑封面缓存

### 6.2 缓存策略
- **内存缓存**：当前播放列表、过滤结果
- **磁盘缓存**：
  - 专辑封面图片（最大500MB，LRU淘汰）
  - 歌词文件（最多1000首）
  - 元数据索引（加速搜索）
- **缓存清理**：用户可配置自动清理周期

### 6.3 异步处理
- 文件扫描使用后台线程，避免UI阻塞
- 进度报告机制（百分比显示）
- 支持取消操作
- 批量操作队列化处理

### 6.4 错误处理
- 文件访问权限错误友好提示
- 损坏文件自动跳过并记录日志
- 网络请求失败重试机制（最多3次）
- 全局异常捕获与用户反馈

### 6.5 配置管理
- JSON格式配置文件
- 自动备份最近5个版本
- 配置验证与默认值回退
- 导入/导出设置功能

## 10. 已知限制与约束

### 10.1 技术限制
- LibVLC依赖，应用体积较大（~50MB）
- 某些DRM保护文件无法播放
- 网络歌词服务依赖第三方API
- Linux平台媒体键支持可能不完整

### 10.2 业务约束
- 纯本地播放器，不内置在线音乐服务
- 不支持流媒体播放
- 不包含社交功能
- 不涉及付费订阅模式

### 10.3 法律合规
- 尊重版权，仅播放用户合法拥有的音乐
- 不破解DRM保护
- 开源许可证兼容（MIT/Apache 2.0）
- 第三方库许可证审查
